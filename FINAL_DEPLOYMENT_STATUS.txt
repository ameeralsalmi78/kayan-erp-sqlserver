🚀 خطوات النشر السريعة:

1️⃣ نشر على Netlify (الطريقة الأسهل):
   
   a) عبر الموقع:
      • اذهب: https://app.netlify.com
      • اضغط: "Add new site" → "Deploy manually"
      • اسحب مجلد "dist" وأفلته
      • ✅ تم!

   b) عبر CLI:
      npm install -g netlify-cli
      netlify login
      netlify deploy --prod --dir=dist

2️⃣ أضف متغيرات البيئة في Netlify:
   • VITE_SUPABASE_URL
   • VITE_SUPABASE_ANON_KEY
   
   احصل عليها من: Supabase Dashboard → Settings → API

3️⃣ فعّل حماية كلمات المرور (مهم!):
   • اذهب: https://app.supabase.com
   • اختر مشروعك
   • Authentication → Settings
   • فعّل: "Enable Leaked Password Protection"
   • احفظ التغييرات
