🎯 كيف تستخدم السورس كود:

1️⃣ حمّل الملف:
   - الملف موجود في المسار أعلاه
   - يمكنك نسخه أو تحميله

2️⃣ فك الضغط:
   tar -xzf kayan-erp-source-code.tar.gz
   cd project

3️⃣ ثبت الحزم:
   npm install

4️⃣ أضف بيانات Supabase في .env:
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_anon_key

5️⃣ شغل المشروع:
   npm run dev

6️⃣ للنشر:
   npm run build
