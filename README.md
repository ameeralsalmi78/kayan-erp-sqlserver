# KAYAN ERP

![KAYAN ERP Logo](public/kayan-erp-1.jpg)

## Enterprise Resource Planning System

A comprehensive ERP system designed for modern businesses.

## Features

- Sales & Purchases Management
- Inventory Control
- Point of Sale (POS)
- Accounting & Finance
- Human Resources
- Reports & Analytics

## Technologies

- React 18 + TypeScript
- Tailwind CSS
- Supabase (PostgreSQL)
- Recharts

## Quick Start

```bash
npm install
npm run dev
```
