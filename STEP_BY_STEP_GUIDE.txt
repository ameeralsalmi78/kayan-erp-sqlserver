🔄 لاستخدام السورس على جهاز آخر:

1. انسخ المشروع كاملاً
2. ثبت الحزم:
   npm install

3. أضف ملف .env مع بيانات Supabase:
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key

4. شغل المشروع:
   npm run dev

5. للنشر:
   npm run build
